# Supremum
real number $s$ is the least upper bound, or supremum, for a set $A \subseteq \mathbb{R}$ if it meets the following two criteria:
1. $s$ is an upper bound for A;
2. if $b$ is any upper bound for $A$, then $s \leq b$
