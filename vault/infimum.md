# infimum
a [[real number]] $s$ is the greatest lower bound, or infimum, of a set $A \subseteq \mathbb{R}$ if 
1. $s$ is an lower bound for $A$, and
2. if $b$ is any lower bound for $A$, then $b \leq s$
